# kr20_icc_kappa_item_analysis
Conducted in Python, this analysis examines the internal consistency (using Kuder-Richardson Formula 20 [KR-20]), inter-rater reliability (using Intraclass Correlation Coefficient [ICC] and Cohen's Kappa), item difficulty and discrimination indices, and the relationship between multiple-choice and essay sections for mixed-format achievement tests.
## 📊 Overview
This script provides a **comprehensive reliability analysis** for mixed-format achievement tests comprising multiple-choice questions (Section A) and essay questions (Section B). The analysis employs multiple psychometric methods to evaluate the consistency, agreement, and discriminative power of test items.
## Analysis Summary
- **KR-20:** 0.81 (Good internal consistency for classroom assessment)
- **ICC(2,k):** 0.95 (Excellent inter-rater agreement between essay markers)
- **Section Correlation:** 0.84 (Strong positive relationship between MCQ and essay scores)
- **Sample Size:** 20 students
---
## Key Features
- **KR-20 Reliability** for dichotomously scored multiple-choice items
- **Inter-Rater Reliability** using ICC and Cohen's Kappa for essay scoring
- **Item Difficulty Analysis** (p-values) for all 20 items
- **Item-Total Correlations** for discrimination indices
- **Section Correlation** between MCQ and essay components
- **Comprehensive Descriptive Statistics** for all sections
- **Excel File Support** for easy data input
- **Detailed Interpretation Guidelines** for each metric
---
## 📈 Methods Used
| Method | Purpose |
|--------|---------|
| **Kuder-Richardson Formula 20 (KR-20)** | Internal consistency of dichotomous items |
| **Intraclass Correlation Coefficient ICC(2,k)** | Inter-rater agreement (average of raters) |
| **Cohen's Kappa** | Category agreement for each essay |
| **Pearson Correlation** | Linear relationship between variables |
| **Item Difficulty (p-value)** | Proportion of correct responses per item |
| **Item-Total Correlation** | Discrimination power of each item |
